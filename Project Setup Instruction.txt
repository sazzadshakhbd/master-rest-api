Eslint Configuration
-------------------
1. Go eslint.org 
2. Copy this code — npm init @eslint/config@latest
3. Run this code our terminal or project
4. Make sure your project is installed latest node version
5. Select ‘to check syntax and find problems’
6. Select ‘Javascript Modules (import and export)
7. Select ‘None of this’
8. Select ‘Yes’
9. Select ‘Node’
10. Select ‘Json’
11. Select ‘Yes’
12. Select ’rpm’
13. Show new file - name ‘.eslintrc.json’ in our project 
14. Commit this on git
15. Eslint plugin install in our vs-code

Setup Express Server
--------------------
1. Install express server - "npm install express"
2. Install typescript dependency  - "npm install -D @types/express"
3. create - src folder > app.ts file in our root project
4. write this code in our app.ts file
   a) import express from 'express' [first of all express import]
   b) const app = express(); [create a app instance]
   c) export default app [exprot express]
5. import express from server.ts file - "import app from './src/app'"
6  server setup code declare function --- 
    const SrartServer = () => {

	const port  = process.env.PORT || 3000;

	app.listen(port, ()=> {
	  console.log(`Listening on port: ${port}`);
	}
    }
    SrartServer();
7. Now go to app.ts file and create route. write this code. get means http methods like [get, post, delete, patch, put]
